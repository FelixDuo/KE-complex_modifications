Left command + hjkl to active windows

Use scripts to activate specific window inside the app that has multiple windows https://trk.tools/sh/keyboard-shortcuts-to-switch-windows-on-macos For example open two terminals and bind with b k and b l and open Brave Browser (by default is binded with u, you can update in data.plist file). Now you can switch between terminal windows and Brave with left_option + k left_option + l or left_option + u.
